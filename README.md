## Contribución al proyecto

¿Te animas a colaborar? Sigue estos pasos:

1. Haz un fork de este repositorio (botón arriba a la derecha en GitHub).
2. Clona tu fork:

git clone git@github.com:tu_usuario/readme-project.git
   
3. Crea una nueva rama:

git checkout -b feature/nueva-seccion

4. Haz tus cambios, confírmelos y sube la rama:
git add .
git commit -m "feat: añade nueva sección"
git push -u origin feature/nueva-seccion


5. En GitHub, abre un *pull request* desde tu rama hacia la rama `main` del repositorio original.
